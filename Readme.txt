#Autor: Cheuk Hang Ku (Cherren)
To run the program, please make sure you downloaded and implemented the external library called "Bouncycastle version - JDK1.8".

This program perfrom the following encryption methods: STS protocol, RSA encryption, 3-DES encryption. Also, Counter (CTR) Mode is implemented.
This program shows that Alice and Bob can send/ receive messages by using 3-DES with CTR mode after the secure session established. Assume that each message will be in 64 bytes.
